3D Triangles → 2D Frames → Deformation Gradient

Project 3D triangles into canonical local frames, then measure how one deforms into another via the per-face Jacobian — the core computation behind ARAP, UV parameterization, and shape registration.

Part 1 — Projecting a Single 3D Triangle into 2D

The Problem

A triangle in 3D space is defined by three vertices — v0, v1, and v2 — at arbitrary positions. The same triangle could appear at many different positions and rotations in the world, yet its shape (relative side lengths and angles) never changes.

To compare or process triangle geometry, we need a canonical 2D representation: a coordinate frame built from the triangle itself, where v0 sits at the origin and the base edge lies along the X-axis. The 2D coordinate of v2 in this frame fully describes the shape — no matter where the triangle was in 3D space.

The Math — Step by Step
1
Build the Local Coordinate Frame

Pin v0 at the origin. The base edge v0→v1 defines the local X-axis. Dividing by its length gives a unit vector $\hat{u}_1$ — our X-axis direction.

$$\vec{u}_1 = v_1 - v_0$$ $$L = \lVert\vec{u}_1\rVert, \qquad \hat{u}_1 = \frac{\vec{u}_1}{L}$$
# v0 and v1 are fixed
v0 = [1.0, 2.0, 1.0]
v1 = [5.0, 2.0, 1.0]
u1 = v1 − v0 = [4.00, 0.00, 0.00]
L = |u1| = 4.000
û1 = u1/L = [1.000, 0.000, 0.000]
2
Project v2 onto the Base — Shadow (X)

Express v2 relative to v0 to get the offset $\vec{u}_2$. The dot product of $\vec{u}_2$ with the unit base $\hat{u}_1$ gives the component that lies along the base — the local X-coordinate.

$$\vec{u}_2 = v_2 - v_0$$ $$x = \vec{u}_2 \cdot \hat{u}_1$$
# v2 moves with the sliders
v2 = [1.0, 5.0, 1.0]
u2 = v2 − v0 = [0.00, 3.00, 0.00]
# dot product = shadow on base axis
x = u2 · û1 = 0.000
3
Perpendicular Remainder — Height (Y)

Subtract the shadow component from $\vec{u}_2$, leaving only the part perpendicular to the base. Its length is the local Y-coordinate — the triangle's height.

$$\vec{u}_{2\perp} = \vec{u}_2 - x\,\hat{u}_1$$ $$y = \lVert\vec{u}_{2\perp}\rVert$$
# remove the base-aligned part
shadow = x · û1 = [0.00, 0.00, 0.00]
u2⊥ = u2 − shadow = [0.00, 3.00, 0.00]
# height = length of remainder
y = |u2⊥| = 3.000

Why Does the Dot Product Give Us the Shadow?

The dot product of any vector $\vec{u}_2$ with a unit vector $\hat{u}_1$ returns the scalar length of $\vec{u}_2$'s shadow cast onto the line defined by $\hat{u}_1$. Formally:

$$x = \vec{u}_2 \cdot \hat{u}_1 = \lVert\vec{u}_2\rVert \cos\theta$$

where $\theta$ is the angle between them. When $\theta = 0°$ the vectors are parallel and $x = \lVert\vec{u}_2\rVert$ (full length). When $\theta = 90°$ the vectors are perpendicular and $x = 0$ (no shadow).

Subtracting this shadow from $\vec{u}_2$ leaves a vector exactly perpendicular to $\hat{u}_1$ — the height component. Together, shadow and height give us a complete, orthogonal decomposition of $\vec{u}_2$ in the triangle's own frame.

Result: Canonical 2D Coordinates

v0′ — always
( 0, 0 )
v1′ — base endpoint
( 4.000, 0 )
v2′ — (shadow, height)
( 0.000, 3.000 )

These three 2D coordinates fully encode the triangle's geometry. Two triangles with the same $v_0',\, v_1',\, v_2'$ are congruent — no matter where they were in 3D space.

Part 2 — Deformation Gradient Between Two Triangles

From Projection to Jacobian

Once a source and a target triangle are each in their own canonical 2D frame, we can encode how one deforms into the other as a 2×2 matrix — the deformation gradient (Jacobian) $J$. This is the per-face computation at the heart of the research code:

$$M_s = \begin{bmatrix} L_s & x_s \\ 0 & y_s \end{bmatrix}, \qquad M_t = \begin{bmatrix} L_t & x_t \\ 0 & y_t \end{bmatrix}, \qquad J = M_t \, M_s^{-1}$$

Columns of $M$ are the 2D edge vectors from $v_0'$: first column $= (L, 0)^\top$, second $= (x, y)^\top$. Because $M$ is upper-triangular, $M^{-1}$ has a closed form and $J$ reduces to $\bigl[\tfrac{L_t}{L_s},\ \tfrac{x_t L_s - L_t x_s}{L_s y_s};\; 0,\ \tfrac{y_t}{y_s}\bigr]$.

Use Cases in Research Practice
UV / Mesh Parameterization

Each 3D triangle face is projected to a 2D UV atlas. J measures per-face texture stretch. Minimizing σ_t across all faces gives an isometric unwrapping — minimal texture distortion, equal texel density everywhere.

Goal: σ_t ≈ 1 uniformly across faces
ARAP Surface Deformation

As-Rigid-As-Possible deformation (the target method in the code) tries to keep J close to a rotation matrix on every face — both singular values ≈ 1. The per-face σ_t is the energy term ARAP minimises.

Ideal: σ_max ≈ σ_min ≈ 1 → J ≈ rotation
Shape Registration / Morphing

When registering ShapeNet source meshes to deformed targets, per-face J reveals which regions compress or stretch most. High σ_t_con faces are "hard" regions where the shape changes character — useful for evaluating method quality.

σ_t_con > 2 → significant local shape change

What Do σ_t and σ_t_con Measure?

The singular values $\sigma_{\max} \ge \sigma_{\min} > 0$ of $J$ are the principal stretch factors: $\sigma_{\max}$ is the maximum stretch direction, $\sigma_{\min}$ the minimum.

$$\sigma_t = \max\!\left(\sigma_{\max},\; \frac{1}{\sigma_{\min}}\right) \;\ge\; 1 \qquad \sigma_{t,\mathrm{con}} = \max\!\left(\frac{\sigma_{\max}}{\bar\sigma},\; \frac{\bar\sigma}{\sigma_{\min}}\right), \quad \bar\sigma = \tfrac{\sigma_{\max}+\sigma_{\min}}{2}$$

$\sigma_t = 1$ iff $J$ is an isometry (rotation/reflection). $\sigma_{t,\mathrm{con}} = 1$ iff $J$ is a conformal map (angle-preserving, possibly scaled). In the research code these are averaged over all faces — weighted or unweighted — to get a global mesh quality score.

Part 3 — ARAP & ACAP: Loss Functions from the Paper

Two Ways to Penalize Shape Distortion

Given the per-face Jacobian $J_f$, both ARAP and ACAP define a loss function as the sum of per-face deformation energies. They differ only in what the ideal map is: a rigid rotation (ARAP) or a conformal scaled rotation (ACAP). Both energies reduce to simple expressions in the singular values $\sigma_{\max}, \sigma_{\min}$ of $J$.

Paper source: Huang, Q.-X., Wicke, M., Adams, B., & Guibas, L. Shape Decomposition using Modal Analysis. Eurographics 2009, Vol. 28, No. 2. [PDF] — adopts the ARAP deformation energy (their Eq. 1, credited to Sorkine & Alexa, SGP 2007) as the foundation for modal analysis of shape deformations.
ARAP
As-Rigid-As-Possible

Find the deformation where each face Jacobian is as close as possible to a rotation matrix R ∈ SO(2). Rotations preserve all lengths and angles: both singular values equal 1.

$$\mathcal{L}_\text{ARAP} = \sum_{f} A_f \underbrace{\min_{R_f \in SO(2)} \|J_f - R_f\|_F^2}_{\text{per-face rigidity loss}}$$ $$= \sum_{f} A_f \bigl[(\sigma_{\max} - 1)^2 + (\sigma_{\min} - 1)^2\bigr]$$

Closed-form optimal rotation: $R_f^* = UV^T$ from $J_f = U\Sigma V^T$. Zero iff $\sigma_{\max} = \sigma_{\min} = 1$ (isometry).

ACAP
As-Conformal-As-Possible

Find the deformation where each face Jacobian is as close as possible to a scaled rotation sR. Scaled rotations preserve angles but allow uniform area change: $\sigma_{\max} = \sigma_{\min}$.

$$\mathcal{L}_\text{ACAP} = \sum_{f} A_f \underbrace{\min_{\substack{s_f > 0 \\ R_f \in SO(2)}} \|J_f - s_f R_f\|_F^2}_{\text{per-face conformal loss}}$$ $$= \sum_{f} A_f \cdot \dfrac{(\sigma_{\max} - \sigma_{\min})^2}{2}$$

Optimal scale: $s_f^* = \tfrac{\sigma_{\max}+\sigma_{\min}}{2}$. Same optimal rotation $R_f^* = UV^T$. Zero iff $\sigma_{\max} = \sigma_{\min}$ (conformal map).

The Paper's Vertex-Based ARAP Energy — Eq. (1)

Huang et al. formulate ARAP over vertex neighborhoods, not individual faces. For a set of points $P = \{p_1 \ldots p_N\}$ with displacements $u_i = p_i' - p_i$:

$$E_P(u) = \sum_{p_i \in P} \min_{c_i} \sum_{j \in \mathcal{N}_i} w_{ij} \,\bigl\|c_i \times (p_i - p_j) - u_i + u_j\bigr\|^2 \tag{Eq. 1, [Huang et al. 2009]}$$

Here $\mathcal{N}_i = \{j : \|p_j - p_i\| < \varepsilon\}$ is the vertex neighborhood, $c_i \in \mathbb{R}^3$ is the optimal linearized rotation axis (so $c_i \times (p_i - p_j)$ approximates $(R_i - I)(p_i - p_j)$), and $w_{ij} = (|\mathcal{N}_i| + |\mathcal{N}_j|)^{-1}$ are symmetrized weights. The minimization over each $c_i$ finds the best-fit local rotation for that vertex's neighborhood.

Connection to the Jacobian: For a single triangular face, this vertex sum collapses to $\mathcal{L}_\text{ARAP}(J_f) = \min_{R \in SO(2)} \|J_f - R\|_F^2 = (\sigma_{\max}-1)^2 + (\sigma_{\min}-1)^2$, where $J_f$ is the per-face 2×2 deformation gradient from Part 2.

The paper uses $E_P$ to build the Hessian $H_P = L \otimes I_3 - BC^{-1}B^T$ (their Eq. 7), whose eigenvectors define the space of typical deformations $\mathcal{U}_{N_t}$ used for shape decomposition.

SVD: The Bridge from J to Both Losses

Every 2×2 Jacobian decomposes as $J = U \Sigma V^T$ where $U, V \in SO(2)$ and $\Sigma = \mathrm{diag}(\sigma_{\max}, \sigma_{\min})$. The same optimal rotation $R^* = UV^T$ solves both inner minimizations; only the scaling differs:

$$\underbrace{\min_{R} \|J - R\|_F^2}_{\text{ARAP}} = \|{\Sigma} - I\|_F^2 = (\sigma_{\max}{-}1)^2 + (\sigma_{\min}{-}1)^2$$ $$\underbrace{\min_{s,R} \|J - sR\|_F^2}_{\text{ACAP}} = \|\Sigma - s^*I\|_F^2 = \frac{(\sigma_{\max}-\sigma_{\min})^2}{2}, \qquad s^* = \frac{\sigma_{\max}+\sigma_{\min}}{2}$$

Geometrically: ARAP penalizes any deviation of either singular value from 1 — it wants the deformation ellipse to be a unit circle. ACAP only penalizes the difference between singular values — it wants the ellipse to be a circle (of any radius). Every isometry is also conformal, so $\mathcal{L}_\text{ARAP} = 0 \Rightarrow \mathcal{L}_\text{ACAP} = 0$, but not vice versa.
